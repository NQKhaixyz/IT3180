Tổng Quan Dự Án

Quản Lý Cư Trú
Hệ thống quản lý cư trú toàn diện với các tính năng:

Quản lý hộ khẩu

Căn hộ, tầng, chủ hộ, số điện thoại, liên hệ khẩn cấp, số chỗ xe, ngày vào ở, trạng thái sở hữu, ngày hết hợp đồng, diện tích.

Quản lý nhân khẩu

Họ tên, ngày sinh, giới tính, CCCD/CMND, loại cư trú (thường trú/tạm trú).

Biến động cư trú

Ghi nhận tạm trú, tạm vắng, chuyển đến, chuyển đi. Timeline theo dõi các biến động.

Quản Lý Thu Phí
Hệ thống thu phí linh hoạt với nhiều loại khoản thu:

Danh mục khoản thu

Phí bắt buộc (bảo vệ, vệ sinh, vận hành, quản lý) và phí tự nguyện (từ thiện, biển đảo). Tính theo m2 hoặc cố định.

Quy trình thu phí

Tạo đợt thu -> Tạo nghĩa vụ -> Thu phí -> In phiếu thu -> Cập nhật công nợ

Thanh toán

Hỗ trợ tiền mặt và chuyển khoản. Theo dõi người nộp, người thu, mã giao dịch ngân hàng.

Bảo Mật & Phân Quyền

Xác thực (Authentication)

Password hash PBKDF2, session token server-side, cookie httpOnly + sameSite=lax, secure flag production, khóa tài khoản sau 5 lần đăng nhập sai.

Phân quyền (RBAC)

3 vai trò: ADMIN (toàn quyền), ACCOUNTANT (thu phí), TEAM_LEADER (cư trú). Phân quyền module-level: SYSTEM, FEE, RESIDENT, REPORT.

Audit Logging

Ghi nhận đầy đủ: đăng nhập/đăng xuất, tạo/cập nhật/xóa, thu phí, khóa/mở khóa tài khoản. Theo dõi actor, action, entity, detail.

Input Validation

API error contract chuẩn hóa, validation email/phone/số, giới hạn pagination, strong password policy, enum validation.

Kiểm Thử & Triển Khai

E2E Testing

Smoke test cho các API critical: authentication, core read APIs, report export, audit log. Command: npm run test:e2e:smoke

Security Testing

Password hashing, session management, RBAC enforcement, input validation, audit logging. Security checklist đầy đủ.

Triển khai

Production build với Next.js, Vercel-compatible, SQLite local dev, PostgreSQL production-ready. Environment variables: DATABASE_URL, ALLOW_BOOTSTRAP_SIGNUP.
